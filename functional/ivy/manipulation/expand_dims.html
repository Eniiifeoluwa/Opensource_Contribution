expand_dimsÔÉÅ

ivy.expand_dims(x, axis=0, out=None)[source]ÔÉÅ

Expands the shape of an array by inserting a new axis with the size of one. This new axis will appear at the axis position in the expanded array shape.

Parameters

  • x (Union[Array, NativeArray]) ‚Äì input array.

  • axis (int) ‚Äì position in the expanded array where a new axis (dimension) of size one will be (default: 0) added. If array x has the rank of N, the axis need to be between [-N-1, N]. Default: 0.

  • out (Optional[Union[Array, NativeArray]]) ‚Äì optional output array, for writing the result to. It must have a shape that the (default: None) inputs broadcast to.

Return type

Array

Returns

ret ‚Äì an array with its dimension added by one in a given axis.

Examples

>>> x = ivy.array([[0, 1], [1, 0]])
>>> y = ivy.expand_dims(x)
>>> print(y)
ivy.array([[[0, 1],
            [1, 0]]])
>>> print(x.shape, y.shape)
(2, 2) (1, 2, 2)
