expand_dims

ivy.expand_dims(x, axis=0, *, out=None)[source]

Expands the shape of an array by inserting a new axis with the size of one. This new axis will appear at the axis position in the expanded array shape.

Parameters

  • x (Union[Array, NativeArray]) – input array.

  • axis (int) – position in the expanded array where a new axis (dimension) of size one will be (default: 0) added. If array x has the rank of N, the axis need to be between [-N-1, N]. Default: 0.

  • out (Optional[Array]) – optional output array, for writing the result to. It must have a shape that the (default: None) inputs broadcast to.

Return type

Array

Returns

  • ret – an array with its dimension added by one in a given axis.

  • This function conforms to the `Array API Standard

  • <https (//data-apis.org/array-api/latest/>`_. This docstring is an extension of the)

  • `docstring <https (//data-apis.org/array-api/latest/API_specification/generated/signatures.manipulation_functions.expand_dims.html>`_ )

  • in the standard. The descriptions above assume an array input for simplicity, but

  • the method also accepts ivy.Container instances in place of

  • ivy.Array or ivy.NativeArray instances, as shown in the type hints

  • and also the examples below.

Functional Examples

With ivy.Array input: >>> x = ivy.array([0, 1, 2]) >>> print(x.shape) (3,)

>>> y = ivy.expand_dims(x)
>>> print(y.shape)
(1, 3)

>>> print(y)
ivy.array([[0, 1, 2]])

>>> x = ivy.array([[0.5, -0.7, 2.4], [1, 2, 3]])
>>> print(x.shape)
(2, 3)

>>> y = ivy.zeros((2, 1, 3))
>>> print(y)
ivy.array([[[0., 0., 0.]],
       [[0., 0., 0.]]])

>>> ivy.expand_dims(x, axis=1, out=y)
>>> print(y)
ivy.array([[[0.5,-0.7,2.4]],[[1.,2.,3.]]])

>>> x = ivy.array([[-1, -2], [3, 4]])
>>> print(x)
ivy.array([[-1, -2],
       [ 3,  4]])

>>> ivy.expand_dims(x, out=x)
>>> print(x)
ivy.array([[[-1,-2],[3,4]]])

With ivy.NativeArray input:

>>> x = ivy.native_array([0, 1, 2])
>>> print(x)
tensor([0,1,2],dtype=torch.int32)

>>> y = ivy.expand_dims(x)
>>> print(y)
ivy.array([[0,1,2]])

>>> print(y)
ivy.array([[0, 1, 2]])

With ivy.Container input: >>> x = ivy.Container(a=ivy.array([0., 1., 2.]), b=ivy.array([3., 4., 5.])) >>> y = ivy.expand_dims(x, axis=-1) >>> print(y) {

a: ivy.array([[0.],

[1.], [2.]]),

b: ivy.array([[3.],

[4.], [5.]])

}

Instance Method Examples

Using ivy.Array instance method: >>> x = ivy.array([0., 1., 2.]) >>> y = x.expand_dims() >>> print(x.shape, y.shape) (3,) (1, 3)

>>> print(y)
ivy.array([[0., 1., 2.]])

Using ivy.Container instance method: >>> x = ivy.Container(a=ivy.array([[0., 1.], [2., 3.]]), b=ivy.array([[4., 5.], [6., 7.]])) >>> print(x) {

a: ivy.array([[0., 1.],

[2., 3.]]),

b: ivy.array([[4., 5.],

[6., 7.]])

}

>>> y = x.expand_dims(axis=1)
>>> print(y)
{
    a: ivy.array([[[0., 1.]],
                  [[2., 3.]]]),
    b: ivy.array([[[4., 5.]],
                  [[6., 7.]]])
}
