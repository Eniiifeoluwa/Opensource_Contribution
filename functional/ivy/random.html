ivy.random_uniform(low=0.0, high=1.0, shape=None, device=None)[source]ÔÉÅ

Draws samples from a uniform distribution. Samples are uniformly distributed over the half-open interval [low, high) (includes low, but excludes high). In other words, any value within the given interval is equally likely to be drawn by uniform.

Parameters

  • low (float) ‚Äì Lower boundary of the output interval. All values generated will be greater than (default: 0.0) or equal to low.

  • high (float) ‚Äì Upper boundary of the output interval. All the values generated will be less (default: 1.0) than high.

  • shape (Optional[Union[int, Tuple[int, ...]]]) ‚Äì If the given shape is, e.g (m, n, k), then m * n * k samples are drawn. (default: None) If size is None (Default), a single value is returned.

  • device (Optional[Device]) ‚Äì device on which to create the array ‚Äòcuda:0‚Äô, ‚Äòcuda:1‚Äô, ‚Äòcpu‚Äô etc. (default: None)
